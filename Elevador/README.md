# Simulação de Elevador em Java

## Descrição
Projeto desenvolvido em Java com o objetivo de praticar Programação Orientada a Objetos (POO).

O sistema simula o funcionamento de um elevador através do terminal.

---

## Funcionalidades

- Definir capacidade máxima do elevador
- Definir total de andares do prédio
- Entrar pessoas no elevador
- Sair pessoas do elevador
- Subir andares
- Descer andares
- Exibir status atual do elevador

---

## Conceitos aplicados

- Classes e Objetos
- Encapsulamento (`private`)
- Métodos
- Estruturas condicionais (`if`)
- Estrutura de repetição (`do-while`)
- Uso da classe `Scanner`

---

## Objetivo

Praticar lógica de programação e os fundamentos de Programação Orientada a Objetos.
